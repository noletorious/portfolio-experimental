# 3D Portfolio Experience

A modern, cutting-edge 3D web experience built with the latest technologies that companies like **Robinhood** and **Tesla** use in production.

## 🎯 Project Goal

Create an immersive, full-screen 3D portfolio experience that showcases advanced front-end development skills using industry-standard technologies. This project demonstrates expertise in modern React patterns, 3D graphics programming, and performance optimization - skills highly valued at top-tier tech companies.

## 🚀 Technology Stack

### Core Technologies

- **React 19** - Latest React with concurrent features
- **TypeScript** - Type-safe development
- **Vite** - Lightning-fast build tool
- **Three.js** - Industry-standard 3D graphics library
- **React Three Fiber (R3F)** - React renderer for Three.js

### Enhancement Libraries

- **@react-three/drei** - Useful helpers and abstractions for R3F
- **@react-three/postprocessing** - Advanced visual effects and post-processing
- **Framer Motion** - Production-ready motion library
- **Tailwind CSS** - Utility-first CSS framework

## 🏃‍♂️ Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
